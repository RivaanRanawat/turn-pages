#Turn Pages

Collection of Book Clubs for students and friends to read books together under a certain period of time

## App Functions

* Authentication
* Creating / Joining Clubs
* In-Club Group Chat
* Admin Function(Selecting next book)
* All Books Read or In Progress
* Book Reviews
